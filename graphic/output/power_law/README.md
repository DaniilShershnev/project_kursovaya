# Каталог графиков для степенной модели тиксотропных сред

Этот каталог содержит графики для математической модели тиксотропных сред со степенными зависимостями вязкости и модуля упругости от степени разрушения структуры.

## Математическая модель

### Определяющие соотношения

- **Модуль упругости**: G(w) = G₀·w^α
- **Вязкость**: η(w) = η₀·w^β
- **Эволюция напряжения**: ṡ = a·w^β - s·w^(β-α)
- **Эволюция структуры**: ẇ = c[1 - w(1 + b·g(s))]

где w - степень разрушения структуры (0 ≤ w ≤ 1), s - напряжение.

### Материальные функции g(s)

В расчетах используются три типа материальных функций:
1. **Экспоненциальная**: g(s) = e^(hs)
2. **Квадратичная**: g(s) = 1 + (hs)²
3. **Линейная**: g(s) = 1 + hs

### Положение равновесия

Равновесное состояние системы определяется условиями:
- s* = a·(w*)^α
- w* = 1/(1 + b·g(s*))

## Ключевые параметры модели

- **a** - коэффициент накопления структуры
- **α** - показатель степени для модуля упругости G(w) = G₀·w^α
- **β** - показатель степени для вязкости η(w) = η₀·w^β
- **c** - характерная скорость изменения структуры
- **b** - коэффициент интенсивности разрушения
- **h** - параметр материальной функции g(s)

## Типы равновесия

Характер поведения решений вблизи равновесия определяется дискриминантом D:
- **D > 0**: устойчивый узел (монотонная релаксация)
- **D < 0**: устойчивый фокус (колебательная релаксация)
- **D = 0**: граничный случай

## Файлы и скрипты

- `power_law_equilibrium.py` - скрипт для вычисления равновесий и построения Рис. 1-8
